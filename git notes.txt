git push-
Git asks internally: Where should I push current branch master?
Right now your local master branch is not linked/tracking any remote branch, so Git says:
fatal: The current branch master has no upstream branch.

Root cause- Right now my local master branch is not linked/tracking any remote branch, so Git says:

fatal: The current branch master has no upstream branch.

Fix- git push -u origin master
What does -u mean?
-u means set upstream.
It tells Git:
My local master branch should track origin/master
After this, Git remembers:
local master  →  origin/master
